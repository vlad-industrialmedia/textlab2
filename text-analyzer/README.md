# Textlab v4

Платформа аудиту якості тексту для SEO, GEO та цитованості ШІ-пошуком.

## Головне у v4
- **Ключ вводиться прямо в застосунку** — жодних змінних Vercel. Обери провайдера (Gemini / OpenRouter / Anthropic / Groq), встав ключ ліворуч.
- Ключ зберігається лише у твоєму браузері (localStorage), на сервер передається тільки для виклику LLM.
- Метрики українською з підказками (наведи курсор) і зонами ризику.
- Клік на метрику з підсвічуванням → на карті виділяються проблемні фрагменти цього типу.
- Збереження/завантаження проєкту (.json) та експорт автономного HTML-звіту з підсвічуванням.

## Провайдери (потрібен лише один ключ)
- **Gemini** — aistudio.google.com → Get API key. Безкоштовно, без картки. За замовчуванням.
- **OpenRouter** — openrouter.ai/keys. Моделі з «:free» безкоштовні.
- **Anthropic** — console.anthropic.com. Платно.
- **Groq** — console.groq.com. Безкоштовно.

## Деплой (GitHub + Vercel)
1. Залий код у репозиторій (папка `text-analyzer`).
2. На Vercel: Add New → Project → обери репозиторій → Framework: Other → Deploy.
3. Якщо файли у вкладеній папці — Settings → Build and Deployment → Root Directory = `text-analyzer` → Save → Redeploy.
4. Змінні оточення **не потрібні** — ключ вводиться в інтерфейсі.

## Безпека
Ключ вводиться у браузері, тож він видимий у браузері. Для Gemini free це не критично (грошей на кону немає). Для платних провайдерів постав spend limit у кабінеті провайдера.

## Структура
- `api/analyze|rewrite|simulate.js` — ендпоінти, приймають `creds` від клієнта.
- `api/_lib/` — сегментація (offset), статистика (burstiness/MTLD/perplexity-проксі), промпти, провайдери LLM.
- `public/js/metrics-meta.js` — назви метрик, підказки, зони.
- `public/js/export.js` — генерація автономного HTML-звіту.
- `public/js/app.js` — вся клієнтська логіка.
